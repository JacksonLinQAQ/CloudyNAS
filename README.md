# CloudyNAS
It is a project that can let you create a simple NAS system on your computer

## Requirement:
- Python 3.10
### modules: 
- flask
- shutill
- datetime
- socket
- os
- json
- random

## Access Method:
- web browser

## Supported Language:
- 繁體中文 (Traditional-Chinese)
